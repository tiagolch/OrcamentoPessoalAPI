# Orcamento Pessoal API

## Instrução

* Criar Virtual Env:
    * ` virtualenv venv `
* Ativar ambiente virtual:
    * ` source venv\bin\activate`
* Instalar requirements.txt
    * ` pip install -r requirements.txt  `
