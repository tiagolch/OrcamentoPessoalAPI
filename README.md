# Orcamento Pessoal API

## Instrução

* Criar Virtual Env:
    * ` python3 -m venv venv. `
* Ativar ambiente virtual:
    * ` source venv\bin\activate`
* Instalar requirements.txt
    * ` pip install -r requirements.txt  `
